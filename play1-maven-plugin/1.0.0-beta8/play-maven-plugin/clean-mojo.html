play:clean

Full name:

com.google.code.maven-play-plugin:play-maven-plugin:1.0.0-beta8:clean

Description:

Clean Play! temporary directories. Directories being cleaned (fully configurable): - "db" - "lib" and "modules" - "logs" - "precompiled" - "test-result" - "tmp"

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.0.0.
  • Binds by default to the lifecycle phase: clean.
