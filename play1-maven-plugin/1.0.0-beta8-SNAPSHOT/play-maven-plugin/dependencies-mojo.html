play:dependencies

Full name:

com.google.code.maven-play-plugin:play-maven-plugin:1.0.0-beta8-SNAPSHOT:dependencies

Description:

Extract project dependencies to "lib" and "modules" directories. It's like Play! framework's "dependencies" command, but uses Maven dependencies, instead of "conf/dependencies.yml" file.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: test.
  • Since version: 1.0.0.

Optional Parameters

Name Type Since Description
dependenciesClean boolean 1.0.0 Should project's "lib" and "modules" subdirectories be cleaned before dependency resolution. If true, dependenciesOverwrite is meaningless.
Default value is: false.
User property is: play.dependenciesClean.
dependenciesOverwrite boolean 1.0.0 Should existing dependencies be overwritten.
Default value is: false.
User property is: play.dependenciesOverwrite.
dependenciesSkip boolean 1.0.0 Skip dependencies extraction.
Default value is: false.
User property is: play.dependenciesSkip.
dependenciesSkipJars boolean 1.0.0 Should jar dependencies be processed. They are necessary for Play! Framework, but not needed for Maven build (Maven uses dependency mechanism).
Default value is: false.
User property is: play.dependenciesSkipJars.
